Show & Tell: Border Radius

The CSS border property rounds the corners of borders, images, divs and more.
The higher the number (in pixles), the more rounded the corners become.
If two values are set, the first applies to top-left and bottom-right corner, the second applies to top-right and bottom-left corner.
Four values apply to the top-left, top-right, bottom-right, bottom-left corner in that order.
Three values: The second value applies to top-right and also bottom-left.
The rounding doesn't have to be perfectly circular, it can be elliptical.
This is done using a slash ("/") between two values.

border-radius: 5px;

border-radius: 20px

border-radius: 50px;

border-radius: 100px;

border-radius: 5px 20px 5px;

border-radius: 5px 50px 5px;

border-radius: 50px 5px 50px;

border-radius: 30px/10px;

border-radius: 10px/30px;

border-radius: 15px 30px 45px 7.5px;
